이 책은 모바일 웹과 모바일 앱을 만들기 용이한 자바스크립트 프레임워크에 대한 이야기를 담고 있습니다. 모바일 시대가 도래하면서 아이폰이나 안드로이드를 주축으로 하는 많은 모바일 앱들이 선보이게 되었습니다. 빠르고 재미있는 앱들이 많은 인기를 누리면서 우리들의 삶을 여유롭고 즐겁게 해주었던 것 같습니다. 하지만, 앱을 만드는 사람들에 입장에서는 즐겁기도 했지만, 고민거리도 많아지게 되었는데요. 이는 아이폰 앱을 만들기 위해서는 objective-c 라는 c 계열의 언어를 새로 배워야 했고, 안드로이드 앱을 만들기 위해서는 java 언어를 배워야 했으며, 같은 os를 사용하는 스마트폰들 끼리도 다른 제조사에서 만들었고, 다른 해상도를 가지고 있어 한번 만들어 놓은 앱을 다른 기기에서 같이 사용하기가 어려웠기 때문입니다. 그래서, 각각의 스마트폰에서 테스트하고 같은 이미지도 다른 해상도와 파일로 만들어야 하는 고민들이 많았지요.
